DOM : Document Object Model

문서의 객체를 지칭하는 용어 (HTML 안에 입력된 객체)

  • html 하위에 있는 모든 태그들(element)/ 속성(attribute) = 프로퍼티(property)
  • element 가지고 올때 : getElementBy** / Id, ClassName, TagName
  • getElementBy : 단일 엘리먼트 리턴
  • byClassName / byTagName : 복수 엘리먼트 리턴
  • HTMLColletion 타입의 유사 배열로 리턴 index 사용하여 선택
