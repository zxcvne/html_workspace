# 공공데이터I front Repository

HTML, CSS, JavaScript 학습 내용을 정리한 저장소입니다.

## 📚 학습 목표

프론트엔드 개발의 기초인 HTML, CSS, JavaScript를 체계적으로 학습하고 실습 예제를 통해 실력을 향상.

## 📝 학습 내용

### HTML

- 기본 구조와 태그
- 시맨틱 HTML5 요소
- 폼과 입력 검증

### CSS

- 선택자와 우선순위
- 박스 모델
- Flexbox레이아웃
- 반응형 디자인 (Media Queries)
- CSS 애니메이션과 전환 효과

### JavaScript

- 변수, 데이터 타입, 연산자
- 함수와 스코프
- 배열과 객체
- DOM 조작과 이벤트 처리
- ES6+ 문법 (화살표 함수, 구조 분해, 스프레드 연산자 등)
- 비동기 프로그래밍 (Promise, async/await)
- Fetch API와 AJAX
